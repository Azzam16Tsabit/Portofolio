IoT Hydroponic

Project Detail

Description

In this project, I designed, estimated the budget, programmed the IoT system, and installed the hydroponics system. The IoT system I created allows the hydroponics system to monitor pH and report it at specific intervals via a Telegram bot. It also provides alerts if the pH rises or falls outside the set limits. This hydroponic system is also integrated with a solar panel system to achieve energy independence with green energy.
